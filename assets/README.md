# assets

Screenshots, banners, and other images used in the README.

Drop your screenshots here as:
- `cover.png` — hero image at the top of the README (recommended: 16:9, ~1920×1080)
- `components.png` — collage showing several components
- `design-system.png` — screenshot of the design-system.html page

To capture clean screenshots:
1. Open `deck.html` in Chrome
2. Press `F11` for full-screen
3. Use Cmd+Shift+4 (macOS) or Win+Shift+S (Windows) to grab the screen
4. Save into this folder
